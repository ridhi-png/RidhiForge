# Ridhi Portfolio (Vite + React)

This folder contains an animated personal portfolio website built with React and Vite.

## Run locally

```bash
cd portfolio
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run preview
```

## Deploy

You can deploy the generated `dist/` folder to any static hosting provider (GitHub Pages, Netlify, Vercel, Render static sites, etc.).

Typical flow:

1. Run `npm run build`.
2. Upload or publish the `dist/` directory on your hosting platform.

For GitHub Pages with Vite, follow: https://vite.dev/guide/static-deploy.html#github-pages
